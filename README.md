# Khedma Releases

This repository hosts public releases for the Khedma NFC Kiosk App.

## Download Latest Release

Visit the [Releases page](https://github.com/najm101/khedma-releases/releases) to download the latest APK files.

## Installation

1. Download the appropriate APK for your device architecture
2. Enable "Install from Unknown Sources" in Android settings
3. Install the APK file
